# 📚 Library Management System (Java Console Application)

A simple **console-based Library Management System** built using Core Java.  
This project demonstrates Object-Oriented Programming, Collections, Streams, Exception Handling, and File Handling with basic data persistence.

---

## 🚀 Features

- Add new books
- Remove books by ID
- Issue books
- Return books
- View all books
- View available books
- Search books by title
- Search books by author
- Sort books by title
- Persistent storage using file handling

---

## 🧠 Concepts Used

This project covers the following Java concepts:

### ✔ Core Java

- OOP (Encapsulation, Classes, Objects)
- Collections Framework (`HashMap`)
- Exception Handling (Custom Exceptions)

### ✔ Java 8 Features

- Streams API
- `filter()`, `forEach()`
- `Collectors.toList()`
- Lambda expressions

### ✔ File Handling

- `BufferedWriter`
- `BufferedReader`
- Persistent storage using structured file format

---

## 💾 Data Persistence

Books are stored in a file (`Books.dat`) using the following format:

id|title|author|isIssued
